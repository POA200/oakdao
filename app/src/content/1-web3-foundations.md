# Web3 Foundations: From Web1 to Stacks

Welcome! This course walks you from the early web to a user-owned, programmable internet powered by Bitcoin and Stacks.

> Tip: Start with the Table of Contents preview on the Dashboard, then continue to this page when ready.

## What you'll learn

- The evolution from Web1 → Web2 → Web3
- Why Web3 matters (ownership, openness, incentives)
- Blockchain fundamentals and Bitcoin's role
- What Stacks is and how it extends Bitcoin
- Tools you can use today and how to stay safe
- Practical steps to get hands-on

## Getting ready

- Create a wallet (Xverse or Leather)
- Set aside a few minutes to explore each section
- Optional: Join community channels to learn faster

Let's begin!
